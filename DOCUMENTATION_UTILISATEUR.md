# Documentation Utilisateur - Distrobox

## 📋 Vue d'ensemble

Distrobox est une application de gestion d'entreprise complète qui permet de gérer :
- 📦 **Inventaire et stocks** dans plusieurs entrepôts
- 💰 **Transactions de vente** avec système de point de vente
- 👥 **Gestion des clients** et suivi des comptes
- 💳 **Recouvrement** et gestion des paiements
- 🏦 **Gestion de caisse** quotidienne
- 👨‍💼 **Utilisateurs** avec gestion des rôles

## 🚀 Démarrage rapide

### Première utilisation
1. **Connexion** : Utilisez vos identifiants fournis par l'administrateur
2. **Dashboard** : Vue d'ensemble de votre activité
3. **Navigation** : Utilisez le menu latéral pour accéder aux modules

### Interface principale
- **Menu de navigation** : À gauche de l'écran
- **Contenu principal** : Zone centrale
- **Notifications** : En haut à droite
- **Profil utilisateur** : Coin supérieur droit

## 📦 Gestion des Stocks

### Consulter l'inventaire
1. Allez dans **Stock → Index**
2. Sélectionnez un entrepôt
3. Consultez les produits disponibles

### Attribuer des produits
1. **Stock → Attribution**
2. Choisissez les produits à attribuer
3. Sélectionnez l'entrepôt de destination
4. Validez l'attribution

### Ajouter des produits
1. **Produits → Index** 
2. Cliquez sur "Nouveau produit"
3. Remplissez les informations
4. Sauvegardez

## 💰 Transactions et Ventes

### Types de transactions
- **AU (Achat UV)** : Achats
- **RU (Retour UV)** : Retours
- **EC (Encaissement)** : Encaissements
- **DC (Décaissement)** : Décaissements
- **DP (Dépenses)** : Dépenses diverses
- **RE (Recouvrement)** : Recouvrements

### Créer une transaction
1. **Transactions → Créer**
2. Choisissez le type de transaction
3. Remplissez les détails
4. Validez et imprimez si nécessaire

### Point de vente quotidien
1. **Transactions → Point du jour**
2. Consultez le résumé quotidien
3. Imprimez le rapport si besoin

## 👥 Gestion des Clients

### Ajouter un client
1. **Clients → Index**
2. "Nouveau client"
3. Remplissez les informations :
   - Nom et prénom
   - Adresse et contacts
   - Informations commerciales
4. Le code client sera généré automatiquement

### Format des codes clients
`CL{MM}{YYYY}SM{XXXXXX}`
- **MM** : Mois de création
- **YYYY** : Année de création  
- **XXXXXX** : Numéro séquentiel

### Modifier un client
1. **Clients → Index**
2. Cliquez sur l'icône "Modifier"
3. Mettez à jour les informations
4. Sauvegardez

## 💳 Recouvrement et Paiements

### Consulter les comptes
1. **Recouvrement → Index**
2. Sélectionnez un client
3. Consultez l'historique des paiements

### Enregistrer un paiement
1. **Recouvrement → Index**
2. Choisissez le client
3. Cliquez "Nouveau paiement"
4. Entrez le montant et les détails
5. Validez le paiement

### Suivi des soldes
1. **Comptes → Index**
2. Consultez les soldes par client
3. Filtrez par période si nécessaire

## 🏦 Gestion de Caisse

### Opérations quotidiennes
1. **Caisses → Index**
2. Sélectionnez la date
3. Consultez les mouvements
4. Ajoutez des entrées/sorties si nécessaire

### Réconciliation
1. Vérifiez les montants
2. Contrôlez avec les documents physiques
3. Validez la journée

## 🔧 Paramètres et Configuration

### Gestion des entrepôts
1. **Entrepôts → Index**
2. Ajoutez/modifiez les entrepôts
3. Définissez les emplacements

### Gestion des utilisateurs
*(Réservé aux administrateurs)*
1. **Utilisateurs → Index**
2. Créez de nouveaux comptes
3. Définissez les rôles et permissions

## 📊 Rapports et Impressions

### Documents disponibles
- **Reçus de transaction** : Impression automatique
- **Points de vente** : Rapport quotidien
- **Bordereaux de versement** : Pour les paiements
- **États de comptes** : Situation client

### Impression
1. Accédez au document souhaité
2. Cliquez sur "Imprimer"
3. Choisissez votre imprimante
4. Validez l'impression

## 🔍 Recherche et Filtres

### Recherche rapide
- Utilisez la barre de recherche en haut
- Recherchez par nom, code, ou référence
- Les résultats s'affichent en temps réel

### Filtres avancés
- **Par date** : Sélectionnez une période
- **Par client** : Filtrez par client spécifique
- **Par type** : Choisissez le type de transaction
- **Par montant** : Définissez une fourchette

## 📱 Raccourcis Clavier

- **Ctrl+S** : Sauvegarder
- **Ctrl+P** : Imprimer
- **Ctrl+F** : Rechercher
- **Échap** : Fermer les pop-ups
- **F5** : Actualiser les données

## 🔒 Sécurité et Bonnes Pratiques

### Connexion
- Utilisez un mot de passe fort
- Ne partagez jamais vos identifiants
- Déconnectez-vous en fin de session

### Sauvegardes
- Les données sont sauvegardées automatiquement
- Vérifiez régulièrement avec l'administrateur
- Signalez tout problème immédiatement

### Confidentialité
- Respectez la confidentialité des données clients
- N'accédez qu'aux informations nécessaires
- Suivez les procédures de votre entreprise

## ❓ Dépannage

### Problèmes courants

**Impossible de se connecter**
- Vérifiez vos identifiants
- Contactez l'administrateur
- Vérifiez votre connexion internet

**Application lente**
- Fermez les onglets/applications inutiles
- Redémarrez l'application
- Contactez le support technique

**Erreur lors de l'impression**
- Vérifiez que l'imprimante est allumée
- Contrôlez les pilotes d'impression
- Testez avec un autre document

**Données manquantes**
- Actualisez la page (F5)
- Vérifiez les filtres appliqués
- Contactez l'administrateur
